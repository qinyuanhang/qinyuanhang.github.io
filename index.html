Reverse Linked List 1&2

Reverse Linked List 1
Reverse a singly linked list.

public class Solution {
    public ListNode reverseList(ListNode head) {

    if(head == null) return null;
    if(head.next == null) return head;

    ListNode pre = head, cur = pre.next, nex;
    //这句话很重要，没有就会死循环：
    pre.next = null;

        while(cur.next != null){
            nex = cur.next;
            cur.next = pre;
            pre = cur;
            cur = nex;
        }
        cur.next = pre;
        return cur;
    }
}

Linked List Cycle I&II

Linked List Cycle
Given a linked list, determine if it has a cycle in it.

Follow up:
Can you solve it without using extra space?

Linked List Cycle II
Given a linked list, return the node where the cycle begins. If there is no cycle, return null.

Note: Do not modify the linked list.

Follow up:
Can you solve it without using extra space?

想法：
1：用快慢指针判断是否有将连接的点；（I 解决）
2：如果有连接的点，把快指针放回开头，快慢指针一步一步走，相遇的点就是开始的点

1 

public class Solution {
    public boolean hasCycle(ListNode head) {
        if(head == null || head.next == null) return false;

        ListNode fast = head;
        ListNode slow = head;

        while(fast.next != null && fast.next.next != null){
            fast = fast.next.next;
            slow = slow.next;

            if(fast == slow) return true;
        }
            return false;
    }
}

2 

public class Solution {
    public ListNode detectCycle(ListNode head) {

        if(head == null || head.next == null) return null;
        ListNode fast = head;
        ListNode slow = head;

        while(fast.next != null && fast.next.next != null){
            fast = fast.next.next;
            slow = slow.next;
            if(fast == slow) break;
        }

        if(fast.next == null || fast.next.next == null) return null;

        fast = head;

        while(fast != slow){
            fast = fast.next;
            slow = slow.next;
        }

        return fast;
    }
}

Reorder List
Given a singly linked list L: L0→L1→…→Ln-1→Ln,
reorder it to: L0→Ln→L1→Ln-1→L2→Ln-2→…

You must do this in-place without altering the nodes’ values.

For example,
Given {1,2,3,4}, reorder it to {1,4,2,3}.

想法：
1：找到中点；
：把中点后面的所有点： ：存到stack里面
：反转 ：merge两个链表在一起

记得吧中点的后面一个设置成null

public class Solution {
    public void reorderList(ListNode head) {

            if(head == null) return;

               ListNode dummy = head;

            ListNode fast = head, slow = head;
        while(fast != null && fast.next != null){
            slow = slow.next;
            fast = fast.next.next;
        }

        Stack<ListNode> stack = new Stack<ListNode>();
        ListNode end = slow;
        slow = slow.next;

        while(slow != null){
            stack.push(slow);
            slow = slow.next;
        }

           end.next = null;

        while(!stack.empty()){
            ListNode temp = stack.pop();
            temp.next = head.next;
            head.next = temp;

Add Two Numbers

Add Two Numbers Total Accepted: 85380 Total Submissions: 421576 My Submissions Question Solution
You are given two linked lists representing two non-negative numbers. The digits are stored in reverse order and each of their nodes contain a single digit. Add the two numbers and return it as a linked list.

Input: (2 -> 4 -> 3) + (5 -> 6 -> 4)
Output: 7 -> 0 -> 8

想法：
1：类似两个字符串相加那道题；
2：注意每一次指针的移动；
3：判断是否有进位；
4：最后一个一个进位要记得考虑；

public class Solution {
    public ListNode addTwoNumbers(ListNode l1, ListNode l2) {
        if(l1 == null) return l2;
        if(l2 == null) return l1;


        int carry = 0, add = 0;
        ListNode dummy = new ListNode(0);
        ListNode head = dummy;


    while(l1 != null && l2 != null){
        add = l1.val + l2.val + carry;
        if(add >= 10){
            carry = 1;
            add = add % 10;
        }else{
            carry = 0;
        }

        head.next = new ListNode(add);
        head = head.next;
        l1 = l1.next;
        l2 = l2.next;
    }

    if(l1 == null){
        while(l2 != null){
            add = l2.val + carry;
            if(add >= 10){
                carry = 1;
                add = add % 10;
            }else{
                carry = 0;
            }
            head.next = new ListNode(add);
            head = head.next;
            l2 = l2.next;
        }
    }else{
         while(l1 != null){
            add = l1.val + carry;
            if(add >= 10){
                carry = 1;
                add = add % 10;
            }else{
                carry = 0;
            }
            head.next = new ListNode(add);
            head = head.next;
            l1 = l1.next;
        }
    }

    if(carry == 1) head.next = new ListNode(1);        
    return dummy.next;
    }
}

Rotate List

Given a list, rotate the list to the right by k places, where k is non-negative.
For example:
Given 1->2->3->4->5->NULL and k = 2,return 4->5->1->2->3->NULL.

思路：
1，找到新的头；
2，找到新的结尾；
3，一前一后接在一起；

public class Solution {
    public ListNode rotateRight(ListNode head, int k) {

        if(head == null) return null;

        int tot = 1;
        ListNode head_1 = head;
        while(head_1.next != null){
            tot++;
            head_1 = head_1.next;
        }
        ListNode newEnd = head;
        k = k % tot;
        if(k == 0) return head;     
        k = tot - k;

        for(int i = 1; i < k; i++){
            newEnd = newEnd.next;
        }
        ListNode newHead = newEnd.next;
        head_1.next = head;
        newEnd.next = null;
        return newHead;
    }
}

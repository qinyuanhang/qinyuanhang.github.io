Number of Islands I/II

Number of Islands I
Given a 2d grid map of ‘1’s (land) and ‘0’s (water), count the number of islands.
An island is surrounded by water and is formed by connecting adjacent lands horizontally or vertically.
You may assume all four edges of the grid are all surrounded by water.

Example 1:
11110
11010
11000
00000
Answer: 1

Example 2:
11000
11000
00100
00011
Answer: 3

public class Solution {
    public int numIslands(char[][] grid) {
        int n = grid.length;

        if (n == 0) {
            return 0;
        }

        int m = grid[0].length;

        if (m == 0) {
            return 0;
        }

        int count = 0;

        for(int i = 0; i < grid.length; i++){
            for(int j = 0; j < grid[0].length; j++){
                if(grid[i][j] == '1'){
                    removeIsland(grid, i, j);
                    count++;
                }
            }
        }
        return count;
    }
    //递归调用的函数，找到一个1，就用广度优先把所有的1变成0
    private void removeIsland(char[][] grid, int x, int y){
        if( grid[x][y] == '1'){
            grid[x][y] = '0';
        }else{
            return;
        }

        if(x - 1 >= 0){
            removeIsland(grid, x - 1, y);
        }

        if(x + 1 < grid.length){
            removeIsland(grid, x + 1, y);
        }

        if(y - 1 >= 0){
            removeIsland(grid, x, y - 1);
        }
        if(y + 1 < grid[0].length){
            removeIsland(grid, x, y + 1);
        }
    }
}

First Missing Positive

First Missing Positive My Submissions Question Solution
Given an unsorted integer array, find the first missing positive integer.

For example,
Given [1,2,0] return 3,
and [3,4,-1,1] return 2.

Your algorithm should run in O(n) time and uses constant space.

想法：

###排列，O(NlogN)

###因为要找positive的数，所以负数就过

###找到第一个正数

####第一个正数要是1

#####就比较下一个数是不是比前一个数大1或者等于前一个数

#####要是一直到最后都没少，就直接返回最后一个数加1

####第一个数要不是1就直接返回一

####这个算法由于有一个排列，所以时间不是O（N）；

public class Solution {
    public int firstMissingPositive(int[] nums) {
        if(nums.length <= 0) return 1;

        int start = 0;
        Arrays.sort(nums);

        for(int i = 0;i < nums.length; i++){
            if(nums[i] >= 1){
                start = i;
                break;
            }
        }

        if(nums[start] != 1) return 1;

            while(start + 1 < nums.length){
                if(nums[start]+1==nums[start+1] || nums[start] == nums[start + 1]){
                    start++;
                }else{
                    return nums[start] + 1;
                }
            }
        return nums[nums.length - 1] + 1;
    }
}

Compare Version Numbers

Compare Version Numbers
Compare two version numbers version1 and version2.
If version1 > version2 return 1, if version1 < version2 return -1, otherwise return 0.

You may assume that the version strings are non-empty and contain only digits and the . character.
The . character does not represent a decimal point and is used to separate number sequences.
For instance, 2.5 is not “two and a half” or “half way to version three”, it is the fifth second-level revision of the second first-level revision.

Here is an example of version numbers ordering:

0.1 < 1.1 < 1.2 < 13.37

想法：
把字符串按照“.”拆开，重头到尾比较数字的大小，只要碰到大的那个就返回对应的结果
注意可能有“1”和“1.0.0.0.1”这样的情况，所以按照长度短的那个来比较，
要是有“1”和“1.0.0.0”这样的贱题目，就要比较长度长的后面的所有数字，要是碰到比0大的，就直接返回对应的结果

*注意

###java中split小数点的时候，要这样split（”\.”）要转义一下。要不然翻个出来的是空的数组

public class Solution {
    public int compareVersion(String version1, String version2) {
        String[] v1 = version1.split("\\.");
        String[] v2 = version2.split("\\.");

        int l = v1.length >= v2.length ? v2.length : v1.length;

        for(int i = 0; i < l; i++){
            if(Integer.parseInt(v1[i]) > Integer.parseInt(v2[i])){
                return 1;
            }else if(Integer.parseInt(v1[i]) < Integer.parseInt(v2[i])){
                return -1;
            }
        }
        int possible = v1.length > v2.length ? 1 : -1;
        String[] longer = v1.length > v2.length ? v1 : v2;
        for(int i = l; i < longer.length; i++){
            if(Integer.parseInt(longer[i]) > 0) return possible;
        }
            return 0;
    }
}

Letter Combinations of a Phone Number

Letter Combinations of a Phone Number My Submissions Question Solution
Given a digit string, return all possible letter combinations that the number could represent.

A mapping of digit to letters (just like on the telephone buttons) is given below.

Input:Digit string “23”
Output: [“ad”, “ae”, “af”, “bd”, “be”, “bf”, “cd”, “ce”, “cf”].

public class Solution {
    String[] chars = {"","","abc","def","ghi","jkl","mno","pqrs","tuv","wxyz"};

    public List<String> letterCombinations(String digits) {
        List<String> res = new ArrayList<String>();
        if(digits.length() <= 0){
            return res;
        }
        StringBuffer s = new StringBuffer();
        help(res, s, digits, 0);
        return res;
    }

    private void help(List<String> res, StringBuffer s, String digits, int start){
        if(s.length() == digits.length()){
            res.add(s.toString());
            return;
        }
        int num = digits.charAt(start) - '0';
        for(int i = 0; i < chars[num].length(); i++){
            s.append(chars[num].charAt(i));
            help(res, s, digits, start + 1);
            s.deleteCharAt(s.length() - 1);
        }
    }
}

postRotate Image

Rotate Image My Submissions Question Solution
Difficulty: Medium
You are given an n x n 2D matrix representing an image.

Rotate the image by 90 degrees (clockwise).

Follow up:
Could you do this in-place?

想法：
第一行变最后一列

public class Solution {
    public void rotate(int[][] matrix) {
           Queue<Integer> queue = new LinkedList<Integer>();
        for(int i = 0; i < matrix.length; i++){
            for(int j = 0; j < matrix[0].length; j++){
                queue.offer(matrix[i][j]);
            }
        }

        for(int i = matrix[0].length - 1; i >= 0; i--){
            for(int j = 0; j < matrix.length; j++){
                matrix[j][i] = queue.poll();
            }
        }
    }
}

Restore IP Addresses My Submissions Question Solution
Difficulty: Medium
Given a string containing only digits, restore it by returning all possible valid IP address combinations.

For example:
Given “25525511135”,

return [“255.255.11.135”, “255.255.111.35”]. (Order does not matter)
想法：
要自己的help方法，每次取一个数字，两个数字和三个数字，
记得判断：不是0开始的，并且小于255的数字

当有四个数段并且原来的字符串也分割完了就是一个合法的ip

public class Solution {
    public List<String> restoreIpAddresses(String s) {
        ArrayList<String> res = new ArrayList<String>();
         ArrayList<String> temp = new ArrayList<String>();
        //不可能成为ip的情况
        if(s.length() < 4 || s.length() > 12){
            return res;
        }        
           help(s,res,temp,0);
           return res;
    }

    private void help(String s, ArrayList<String> res, ArrayList<String> temp, int start){
        if(temp.size() == 4){
            //不符合
            if(start != s.length()){
                return;
            }
            StringBuffer bur = new StringBuffer();
            for(String i : temp){
                bur.append(i);
                bur.append(".");
            }
            bur.deleteCharAt(bur.length()-1);
            res.add(bur.toString());
            return;
        }

        for(int i=start; i<s.length() && i<= start+3; i++){
            String tmp = s.substring(start, i+1);
            if(isvalid(tmp)){
                temp.add(tmp);
                help(s,res, temp, i+1);
                temp.remove(temp.size()-1);
            }
        }
        }

    private boolean isvalid(String s){
           if(s.charAt(0) == '0')
            return s.equals("0"); // to eliminate cases like "00", "10"
        int digit = Integer.valueOf(s);
        return digit >= 0 && digit <= 255;
    }
}

postEvaluate Reverse Polish Notation

Evaluate Reverse Polish Notation
Evaluate the value of an arithmetic expression in Reverse Polish Notation.
Valid operators are +, -, *, /. Each operand may be an integer or another expression.

Some examples:
[“2”, “1”, “+”, “3”, ““] -> ((2 + 1) 3) -> 9
[“4”, “13”, “5”, “/“, “+”] -> (4 + (13 / 5)) -> 6

想法：
用栈的方法从前到后 注意string 和 int时间的转换

public class Solution {
    public int evalRPN(String[] tokens) {
        if(tokens.length == 0) return 0;

        Stack<String> stack = new Stack<String>();

        for(int i = 0; i < tokens.length; i++){

            if(!tokens[i].equals("+") && !tokens[i].equals("-") && !tokens[i].equals("*") && !tokens[i].equals("/") ){
                    stack.push(tokens[i]);
            }else{
                int post = Integer.parseInt(stack.pop());
                int pre = Integer.parseInt(stack.pop());
                int result = 0;
            switch(tokens[i]){

                case "+": result = post+pre;
                        break;

                case "-": result = pre-post;
                        break;

                case "*": result = post*pre;
                        break;             

                case "/": result = pre/post;
                        break;             
            }

            stack.push(result+"");
            }
        }    
        return Integer.parseInt(stack.pop());
    }
}

例子：
1 2 3 4
5 6 7 8

打印：
1

public class PrintDia {

    public static void main(String[] args) {
    // TODO Auto-generated method stub        
        int[][] r = {
                {1,2,3},
                {4,5,6},
                {7,8,9},
                        };
        printDia(r);

    public static void printDia(int[][] matrix){
    //first line
        for(int i = 0; i < matrix[0].length; i++){
            helpPrint(0,i, matrix);

            j--;
            i++;
        }        
    }

strstr

Implement strStr();

Returns the index of the first occurrence of needle in haystack, or -1 if needle is not part of haystack.

想法：

  • 最简单的是用substring和equals
  • 有一些变态的公司，某pock..不让用substring，就用复杂的方法，N^2的复杂度
  • 想要高端一点，就用kmp算法

一般方法：

public class Solution {
    public int strStr(String haystack, String needle) {
        if(needle.length() == 0) return 0;
        for(int i = 0; i < haystack.length(); i ++){
            if(i + needle.length() > haystack.length()) break;//容易忘记这个判断，超出范围
            if(haystack.charAt(i) == needle.charAt(0)){
                int sp = i;                    
                for(int j = 0; j < needle.length(); j++){
                    if(needle.charAt(j) == haystack.charAt(sp)){
                        sp++;
                        if(j == needle.length() - 1){
                            return i;
                        }
                    }else{
                        break;
                    }
                }
            }
        }
            return -1;
    }
}

public class KMP {
    //test cases
    public static void main(String[] arge){    
        int i = getIndexKMP("ab", "bllbcb");
        System.out.println(i);        
    }

    /** 获取字符串s的next[]数组 */
    public static int[] getNextSeq(String s) {
        int size = s.length();
        int i = 0;
        int j = -1;
        int[] next = new int[size];
        next[0] = -1;
        while (i < size - 1) {
            if ((j == -1) || (s.charAt(i) == s.charAt(j))) {
                j++;
                i++;
                next[i] = j;
            } else {
                j = next[j];
            }
        }

        for (int k = 0; k <= size - 1; k++) {
            System.out.print(next[k]);
            System.out.print(", ");
        }
        System.out.println();
        return next;
    }

    /**获取index*/
    public static int getIndexKMP(String s, String t) {
        int sSize = s.length();
        int tSize = t.length();
        int i = 0;
        int j = 0;
        int[] next = getNextSeq(t);
        while (i < sSize && j < tSize) {
            if ((j == -1) || (s.charAt(i) == t.charAt(j))) {
                if (j > -1) {
                    System.out.println(s.charAt(i) + "=" + t.charAt(j));
                }
            i++;
            j++;
            } else {
                j = next[j];
            }
        }

        if (j >= tSize) {
            return i - tSize;
        }
        return -1;
    }
}

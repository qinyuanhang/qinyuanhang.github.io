strstr

Implement strStr();

Returns the index of the first occurrence of needle in haystack, or -1 if needle is not part of haystack.

想法：

  • 最简单的是用substring和equals
  • 有一些变态的公司，某pock..不让用substring，就用复杂的方法，N^2的复杂度
  • 想要高端一点，就用kmp算法

一般方法：

public class Solution {
    public int strStr(String haystack, String needle) {
        if(needle.length() == 0) return 0;
        for(int i = 0; i < haystack.length(); i ++){
            if(i + needle.length() > haystack.length()) break;//容易忘记这个判断，超出范围
            if(haystack.charAt(i) == needle.charAt(0)){
                int sp = i;                    
                for(int j = 0; j < needle.length(); j++){
                    if(needle.charAt(j) == haystack.charAt(sp)){
                        sp++;
                        if(j == needle.length() - 1){
                            return i;
                        }
                    }else{
                        break;
                    }
                }
            }
        }
            return -1;
    }
}

public class KMP {

    /** 获取字符串s的next[]数组 */
    public static int[] getNextSeq(String s) {
        int size = s.length();
        int i = 0;
        int j = -1;
        int[] next = new int[size];
        next[0] = -1;
        while (i < size - 1) {
            if ((j == -1) || (s.charAt(i) == s.charAt(j))) {
                j++;
                i++;
                next[i] = j;
            } else {
                j = next[j];
            }
        }

        for (int k = 0; k <= size - 1; k++) {
            System.out.print(next[k]);
            System.out.print(", ");
        }
        System.out.println();
        return next;
    }

    /**获取index*/
    public static int getIndexKMP(String s, String t) {
        int sSize = s.length();
        int tSize = t.length();
        int i = 0;
        int j = 0;
        int[] next = getNextSeq(t);
        while (i < sSize && j < tSize) {
            if ((j == -1) || (s.charAt(i) == t.charAt(j))) {
                if (j > -1) {
                    System.out.println(s.charAt(i) + "=" + t.charAt(j));
                }
            i++;
            j++;
            } else {
                j = next[j];
            }
        }

        if (j >= tSize) {
            return i - tSize;
        }
        return -1;
    }
}

Partition List

Partition List Total
Given a linked list and a value x, partition it such that all nodes less than x come before nodes greater than or equal to x.

You should preserve the original relative order of the nodes in each of the two partitions.

For example,
Given 1->4->3->2->5->2 and x = 3,
return 1->2->2->4->3->5.

想法：

  • 一个简单的问题是用两个queue来分别装合适的node，最后在从小接到大（空间不够）
  • 开另外一个list，把大的接到这个list里面，最后把剩下的两端接起来。
public class Solution {
    public ListNode partition(ListNode head, int x) {

        if(head == null || head.next == null) return head;

        ListNode big = new ListNode(0);
        ListNode dummy_big = big;

        ListNode dummy_small = new ListNode(0);
        dummy_small.next = head;

        head = dummy_small;

        while(head.next != null){
            if(head.next.val >= x){
                big.next = head.next;
                head.next = head.next.next;
                big = big.next;
                big.next = null;//这一步很重要，第二个list的最后一个一定是null'
            }else{
                head = head.next;
            }    
        }

        head.next = dummy_big.next;
        return dummy_small.next;
    }
}

Palindrome Linked List

Palindrome Linked List
Given a singly linked list, determine if it is a palindrome.

Follow up:
Could you do it in O(n) time and O(1) space?

想法：

  • 简单的算法是用stack重头到尾存一遍，再从头到尾和stack里面pop出来的对比，都相等就可以，空间O（n） 和 时间O（n）
  • 复杂的是快慢指针找中点，后半段反过来，前半段和后半段一个一个对。
public class Solution {
    public boolean isPalindrome(ListNode head) {

        if(head == null || head.next==null)
            return true;

        //find list center
        ListNode fast = head;
        ListNode slow = head;

        while(fast.next!=null && fast.next.next!=null){
            fast = fast.next.next;
            slow = slow.next;
        }

        ListNode secondHead = slow.next;
        slow.next = null;

        //reverse second part of the list
        ListNode p1 = secondHead;
        ListNode p2 = p1.next;

        while(p1!=null && p2!=null){
            ListNode temp = p2.next;
            p2.next = p1;
            p1 = p2;
            p2 = temp;
        }

        secondHead.next = null;

        //compare two sublists now
        ListNode p = (p2==null?p1:p2);
        ListNode q = head;
        while(p!=null){
            if(p.val != q.val)
            return false;

            p = p.next;
            q = q.next;
        }

        return true;

    }
}

Remove Duplicates from Sorted List 1&2

Remove Duplicates from Sorted List
Given a sorted linked list, delete all duplicates such that each element appear only once.

For example,
Given 1->1->2, return 1->2.
Given 1->1->2->3->3, return 1->2->3.

public class Solution {
    public ListNode deleteDuplicates(ListNode head) {

    if(head == null) return null;

    ListNode dummy = new ListNode(0);
    dummy.next = head;

    while(head.next != null){
        if(head.next.val == head.val){
            head.next = head.next.next;
        }else{
            head = head.next;
        }
    }
    return dummy.next;
    }
}

Remove Duplicates from Sorted List II
Given a sorted linked list, delete all nodes that have duplicate numbers, leaving only distinct numbers from the original list.

For example,
Given 1->2->3->3->4->4->5, return 1->2->5.
Given 1->1->1->2->3, return 2->3.

  • 思路：
    • 用dummy note
    • 一次检查后两个的val，因为是sorted的，要有重复只能是相邻
    • 用一个int来存已经重复的val
    • 用一个while一直找到下一个不是重复的值为止
    • 要注意检查是不是null，因为检查的是下两个
public class Solution {
    public ListNode deleteDuplicates(ListNode head) {
        ListNode dummy = new ListNode(Integer.MAX_VALUE);
        dummy.next = head;
        head = dummy;

        while(head.next != null && head.next.next != null){
            if(head.next.val == head.next.next.val){
                int same = head.next.val;
                ListNode next = head.next;
                while(next != null && next.val == same){
                    next = next.next;
                }
                head.next = next;
            }else{
                head = head.next;
            }
        }
    return dummy.next;
    }
}

Merge Sorted Lists

Merge Two Sorted Lists
Merge two sorted linked lists and return it as a new list. The new list should be made by splicing together the nodes of the first two lists.

    public class Solution {
        public ListNode mergeTwoLists(ListNode l1, ListNode l2) {
        if(l1 == null) return l2;
        if(l2 == null) return l1;
        ListNode dummy = new ListNode(0);
        ListNode res = dummy;

        while(l1 != null && l2 != null){
            if(l1.val <= l2.val){
                dummy.next = l1;
                l1 = l1.next;
            }else{
                dummy.next = l2;
                l2 = l2.next;
            }
            dummy = dummy.next;
        }

        if(l1 == null){
            dummy.next = l2;
        }else{
            dummy.next = l1;
        }

        return res.next;
    }
}

Merge k Sorted Lists
Merge k sorted linked lists and return it as one sorted list. Analyze and describe its complexity.

  • 想法：
    • 用PriorityQueue，取queue中最小的那个
      *记得要改写Comparator

用法示范:

PriorityQueue<ListNode> queue = new PriorityQueue<ListNode>('queue的size', new Comparator(){
    public int compare(ListNode l1, ListNode l2){
        return l1.val - l2.val;
    }
});

public class Solution {
    public ListNode mergeKLists(ListNode[] lists) {
        if(lists.length == 0) return null;

        PriorityQueue<ListNode> queue = new PriorityQueue<ListNode>(lists.length, new Comparator<ListNode>(){
                public int compare(ListNode l1, ListNode l2){
                    return l1.val - l2.val;
                }
        });

        for(int i = 0; i < lists.length; i++){
            if(lists[i] != null) queue.add(lists[i]);
        }

        ListNode head = new ListNode(0);
        ListNode p = head;

        while(!queue.isEmpty()){
            ListNode temp = queue.poll();
            p.next = temp;
            if(temp.next != null){
                queue.add(temp.next);
            }
            p = p.next;
        }
    return head.next;
    }
}

Reverse Linked List 1&2

Reverse Linked List 1
Reverse a singly linked list.

public class Solution {
    public ListNode reverseList(ListNode head) {

    if(head == null) return null;
    if(head.next == null) return head;

    ListNode pre = head, cur = pre.next, nex;
    //这句话很重要，没有就会死循环：
    pre.next = null;

        while(cur.next != null){
            nex = cur.next;
            cur.next = pre;
            pre = cur;
            cur = nex;
        }
        cur.next = pre;
        return cur;
    }
}

Reverse Linked List II
Reverse a linked list from position m to n. Do it in-place and in one-pass.

For example:
Given 1->2->3->4->5->NULL, m = 2 and n = 4,

return 1->4->3->2->5->NULL.

想法：

  • 找到m前一个位置
  • 把他之后的n - m + 1 个node存到stack里面
  • 再一个一个pop出来，有反转的效果

Note:
Given m, n satisfy the following condition:
1 ≤ m ≤ n ≤ length of list.

    public class Solution {
        public ListNode reverseBetween(ListNode head, int m, int n) {

        if(head == null) return null;
        ListNode dummy = new ListNode(0);
        dummy.next = head;
        ListNode start = dummy;
        Stack<ListNode> stack = new Stack<ListNode>();

        for(int i = 0; i < m - 1; i++){
            start = start.next;
        }

        ListNode toPush = start.next;

        for(int i = 0; i < n - m + 1; i++){
            stack.push(toPush);
            toPush = toPush.next;
        }

        while(!stack.empty()){
            ListNode temp = stack.pop();
            start.next = temp;
            start = temp;
        }

        start.next = toPush;
        return dummy.next;   
    }
}

Linked List Cycle I&II

Linked List Cycle
Given a linked list, determine if it has a cycle in it.

Follow up:
Can you solve it without using extra space?

Linked List Cycle II
Given a linked list, return the node where the cycle begins. If there is no cycle, return null.

Note: Do not modify the linked list.

Follow up:
Can you solve it without using extra space?

想法：
1：用快慢指针判断是否有将连接的点；（I 解决）
2：如果有连接的点，把快指针放回开头，快慢指针一步一步走，相遇的点就是开始的点

1 

public class Solution {
    public boolean hasCycle(ListNode head) {
        if(head == null || head.next == null) return false;

        ListNode fast = head;
        ListNode slow = head;

        while(fast.next != null && fast.next.next != null){
            fast = fast.next.next;
            slow = slow.next;

            if(fast == slow) return true;
        }
            return false;
    }
}

2 

public class Solution {
    public ListNode detectCycle(ListNode head) {

        if(head == null || head.next == null) return null;
        ListNode fast = head;
        ListNode slow = head;

        while(fast.next != null && fast.next.next != null){
            fast = fast.next.next;
            slow = slow.next;
            if(fast == slow) break;
        }

        if(fast.next == null || fast.next.next == null) return null;

        fast = head;

        while(fast != slow){
            fast = fast.next;
            slow = slow.next;
        }

        return fast;
    }
}

Reorder List
Given a singly linked list L: L0→L1→…→Ln-1→Ln,
reorder it to: L0→Ln→L1→Ln-1→L2→Ln-2→…

You must do this in-place without altering the nodes’ values.

For example,
Given {1,2,3,4}, reorder it to {1,4,2,3}.

Add Two Numbers

Add Two Numbers Total Accepted: 85380 Total Submissions: 421576 My Submissions Question Solution
You are given two linked lists representing two non-negative numbers. The digits are stored in reverse order and each of their nodes contain a single digit. Add the two numbers and return it as a linked list.

Input: (2 -> 4 -> 3) + (5 -> 6 -> 4)
Output: 7 -> 0 -> 8

想法：
1：类似两个字符串相加那道题；
2：注意每一次指针的移动；
3：判断是否有进位；
4：最后一个一个进位要记得考虑；

public class Solution {
    public ListNode addTwoNumbers(ListNode l1, ListNode l2) {
        if(l1 == null) return l2;
        if(l2 == null) return l1;


        int carry = 0, add = 0;
        ListNode dummy = new ListNode(0);
        ListNode head = dummy;


    while(l1 != null && l2 != null){
        add = l1.val + l2.val + carry;
        if(add >= 10){
            carry = 1;
            add = add % 10;
        }else{
            carry = 0;
        }

        head.next = new ListNode(add);
        head = head.next;
        l1 = l1.next;
        l2 = l2.next;
    }

    if(l1 == null){
        while(l2 != null){
            add = l2.val + carry;
            if(add >= 10){
                carry = 1;
                add = add % 10;
            }else{
                carry = 0;
            }
            head.next = new ListNode(add);
            head = head.next;
            l2 = l2.next;
        }
    }else{
         while(l1 != null){
            add = l1.val + carry;
            if(add >= 10){
                carry = 1;
                add = add % 10;
            }else{
                carry = 0;
            }
            head.next = new ListNode(add);
            head = head.next;
            l1 = l1.next;
        }
    }

    if(carry == 1) head.next = new ListNode(1);        
    return dummy.next;
    }
}

Android基础知识

Android 的四大核心组件：

  • 1：Activity
  • 2：Service
  • 3：Broadcast Receiver
  • 4：Content Provider

Activity

与用户交互的基本图形单元

有3种状态： 运行，暂停和停止。
生命周期

1: oncreate
2: onstart
3: onresume
4: onpause -> 结束暂停会跳到 onresume
5: onstop -> 又打开这个程序会通过 7: onreset 跳到 onrsume
6: ondestroy( 彻底被关闭)

常见操作

1：按一个按钮打开另外一个 activity

  • private 一个 button， 用findviewbyid找到这个按钮的id
  • 这个button。setonclicklistener（new View.onclicklistener(){
    在这里重写onclick（）{
    Intent i= new Intent(Mainact, act2);这里是在mainact中打开act2
    startactivity(i)；

//如果你是要从act2拿回数据，那就要用startactivityforresult
}
}）

2: 两个act 之间传送数据

  • 用intent.putextra(“name”,”hshshs”);
  • 用bundle，在bundle里面放复杂的数据，再通过inten把bundle传过去；

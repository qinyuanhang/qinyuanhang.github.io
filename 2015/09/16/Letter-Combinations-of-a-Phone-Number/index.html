Letter Combinations of a Phone Number

Letter Combinations of a Phone Number My Submissions Question Solution
Given a digit string, return all possible letter combinations that the number could represent.

A mapping of digit to letters (just like on the telephone buttons) is given below.

Input:Digit string “23”
Output: [“ad”, “ae”, “af”, “bd”, “be”, “bf”, “cd”, “ce”, “cf”].

public class Solution {
    String[] chars = {"","","abc","def","ghi","jkl","mno","pqrs","tuv","wxyz"};

    public List<String> letterCombinations(String digits) {
        List<String> res = new ArrayList<String>();
        if(digits.length() <= 0){
            return res;
        }
        StringBuffer s = new StringBuffer();
        help(res, s, digits, 0);
        return res;
    }

    private void help(List<String> res, StringBuffer s, String digits, int start){
        if(s.length() == digits.length()){
            res.add(s.toString());
            return;
        }
        int num = digits.charAt(start) - '0';
        for(int i = 0; i < chars[num].length(); i++){
            s.append(chars[num].charAt(i));
            help(res, s, digits, start + 1);
            s.deleteCharAt(s.length() - 1);
        }
    }
}

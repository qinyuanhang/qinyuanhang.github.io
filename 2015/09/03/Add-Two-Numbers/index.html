Add Two Numbers

Add Two Numbers Total Accepted: 85380 Total Submissions: 421576 My Submissions Question Solution
You are given two linked lists representing two non-negative numbers. The digits are stored in reverse order and each of their nodes contain a single digit. Add the two numbers and return it as a linked list.

Input: (2 -> 4 -> 3) + (5 -> 6 -> 4)
Output: 7 -> 0 -> 8

想法：
1：类似两个字符串相加那道题；
2：注意每一次指针的移动；
3：判断是否有进位；
4：最后一个一个进位要记得考虑；

public class Solution {
    public ListNode addTwoNumbers(ListNode l1, ListNode l2) {
        if(l1 == null) return l2;
        if(l2 == null) return l1;


        int carry = 0, add = 0;
        ListNode dummy = new ListNode(0);
        ListNode head = dummy;


    while(l1 != null && l2 != null){
        add = l1.val + l2.val + carry;
        if(add >= 10){
            carry = 1;
            add = add % 10;
        }else{
            carry = 0;
        }

        head.next = new ListNode(add);
        head = head.next;
        l1 = l1.next;
        l2 = l2.next;
    }

    if(l1 == null){
        while(l2 != null){
            add = l2.val + carry;
            if(add >= 10){
                carry = 1;
                add = add % 10;
            }else{
                carry = 0;
            }
            head.next = new ListNode(add);
            head = head.next;
            l2 = l2.next;
        }
    }else{
         while(l1 != null){
            add = l1.val + carry;
            if(add >= 10){
                carry = 1;
                add = add % 10;
            }else{
                carry = 0;
            }
            head.next = new ListNode(add);
            head = head.next;
            l1 = l1.next;
        }
    }

    if(carry == 1) head.next = new ListNode(1);        
    return dummy.next;
    }
}
